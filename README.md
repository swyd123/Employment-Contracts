# Employment Contract Generator

An application for generating employment contracts with automatic compliance checking against current legislation and awards.

## Features
- User authentication
- Contract template management
- Automatic compliance checking
- Award integration
- PDF generation

## Tech Stack
- Frontend: React
- Backend: Supabase
- Authentication: Supabase Auth
- Database: PostgreSQL (Supabase)
- PDF Generation: jsPDF

## Setup Instructions

### Prerequisites
- Node.js (v16+)
- Supabase account
- GitHub account
